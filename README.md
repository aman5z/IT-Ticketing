Detailed Description
This project is a browser-based IT ticket management system designed to streamline internal technical support workflows. The system allows users to submit issues while technicians can manage, assign, and resolve support tickets through a centralized dashboard.

The application includes role-based access control, enabling different permissions for users and technicians. Users can create support requests, attach files, and track the status of their tickets, while technicians can assign tickets, update statuses, and close resolved issues.

A dashboard provides real-time statistics such as open tickets, closed tickets, overdue tickets, and total requests, helping administrators quickly monitor system performance and workload.

The backend is powered by Google Apps Script, allowing lightweight serverless data handling and API communication without requiring a traditional server infrastructure.

The interface is fully responsive and includes features such as dark/light themes, live clock, weather display, automatic ticket refresh, and search filtering.
